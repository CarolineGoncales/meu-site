Fundamentos de Gestão de Projetos

Antes de começar:

Como usamos exemplos simples no vídeo, vale uma anotação oficial:

  • Projeto → cria algo específico
  • Programa → organiza vários projetos relacionados
  • Portfólio → reúne projetos e programas alinhados à estratégia da organização

Exemplo:

  • Portfólio: estratégia digital da empresa
  • Programa: transformação digital
  • Projetos:
  • • criar aplicativo
  • • implementar CRM
  • • novo site
  • • automação de marketing


Seu Primeiro Projeto Python

Antes de começar:

Python é uma linguagem de programação de alto nível, muito utilizada em automação, análise de dados, desenvolvimento web e inteligência artificial.

  • Variáveis → armazenam informações
  • Entrada de dados → interação com o usuário
  • Condicionais → tomada de decisão (if/else)
  • Lógica → base de toda programação

Exemplo simples:

  • Receber nome do usuário
  • Validar idade
  • Exibir mensagem personalizada
